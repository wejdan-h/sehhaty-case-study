# Sehhaty Application Case Study

A case study that analyzes the Sehhaty application, focusing on its telemedicine services, user interface design, usability, and its impact on healthcare delivery in Saudi Arabia.

## Overview

The study explores how the Sehhaty application improves access to healthcare services through digital technologies. It examines the application's features, interface design, user experience, and the factors that contributed to its widespread adoption.

## Objectives

- Analyze the healthcare services provided by Sehhaty.
- Evaluate the application's user interface and navigation.
- Identify the reasons behind its popularity in Saudi Arabia.
- Explore the role of telemedicine in improving healthcare accessibility.
